# NLPplayground

## setup
clone repo
```bash
git clone https://github.com/alessioC42/NLPplayground.git
```

change dir
```bash
cd  NLPplayground/webserver
```

configure webserver (set correct path)
```bash
nano config.json
```

install node modules
```bash
npm i
```

execute
```bash
npm run default
```
